# suzuki-ml-literature

Phase 1 goal: learn a reproducible ML workflow using open-access Suzuki reaction literature data.
- Extract reaction data from 2 wet-lab Suzuki papers (training) + 1 paper (validation)
- Reproduce and run a published Suzuki ML model
- Document environment setup, data extraction, and results clearly on GitHub

## Environment
- Mac: Apple Silicon (arm64)
- Python: 3.10 (conda env: `suzuki_ml`)
